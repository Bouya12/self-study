# FuriganaJS

JavaScript学習

## 学習方法

* Chapter1は基本的なJavaScriptの解説のため、ファイル作成無し
* Chapter2,3は基本的なプログラミングの解説メインのため、章最後のまとめの演習問題を作成、保存する
* Chapter4,5はJavaScriptのプログラミングの書き方解説が中心となるため、各解説PGを作成、保存する
