# MarkdownWriting

Markdown学習

## 学習方法

* 書籍を参考にMarkdownでファイル書いていく
* VSCode をエディターとして使用し、プレビューを確認しながら作業する
* VSCode には拡張機能の「Markdown All in One」「Markdown Lint」をインストール
